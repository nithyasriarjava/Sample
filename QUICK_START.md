# 🚀 Quick Start Guide - Chemistry App

## Running the Application

### Prerequisites
- Node.js installed
- npm or yarn package manager

### Installation & Run
```bash
# Navigate to project directory
cd chemistry-app-deploy

# Install dependencies (if not already installed)
npm install

# Start development server
npm run dev
```

The app will open at `http://localhost:5173` (or the port shown in terminal)

## 🐛 Troubleshooting

### If styles don't load:
1. Clear browser cache (Ctrl+Shift+R or Cmd+Shift+R)
2. Restart dev server
3. Check console for errors

### If animations are choppy:
1. Close other browser tabs
2. Check browser hardware acceleration is enabled
3. Update browser to latest version

### If responsive design issues:
1. Test in browser DevTools responsive mode
2. Try different device presets
3. Check viewport meta tag in index.html
